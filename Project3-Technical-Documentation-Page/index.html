Introduction

Python is a versatile, cross-platform, object-oriented programming language known for its simplicity and lightweight nature. Just like JavaScript, Python can be used in various host environments, providing programmatic control over the objects within those environments.

Python offers the following advantages:

  • Versatility: Python can be used in a variety of domains, including web development, scientific computing, data analysis, machine learning, and more. Its flexibility allows developers to tackle diverse projects with a single language.
  • Readability and simplicity: Python's clean and intuitive syntax promotes readability, making it easier for developers to understand and maintain code. The language's design philosophy emphasizes simplicity and reducing the complexity of programming tasks.

Variables

In Python, variables serve as symbolic names for values in your code. When naming variables in Python, you need to follow specific rules and conventions. Variable names must begin with a letter (a-z, A-Z) or an underscore (_). Subsequent characters in the variable name can be letters, digits, or underscores. Python is case-sensitive, meaning that uppercase and lowercase letters are considered distinct. This allows you to have variables like "myVar" and "myvar" as separate entities.

Python also allows the use of Unicode characters in variable names, giving you the flexibility to include non-ASCII characters such as å or ü. However, it's important to note that using non-ASCII characters in variable names might reduce the code's portability and compatibility across different environments or editors.

When choosing variable names in Python, it's good practice to avoid using special characters like @, #, or $. Stick to using letters, digits, and underscores to ensure the name's validity. Additionally, Python follows a naming convention known as snake_case, where multiple words in a variable name are separated by underscores. For example, my_variable_name is more readable than myvariablename.

Variable scope

In Python, variable scope works differently compared to JavaScript. When you declare a variable within a function, it is considered a local variable and is only accessible within that function's scope. Conversely, when you declare a variable outside of any function, it becomes a global variable and is accessible to the entire document or module.

For example, consider the following Python code:

x = 5 def my_function(): y = 10 print(x) # Accessing global variable print(y) # Accessing local variable my_function() print(x) # Accessing global variable outside the function

In this code, we have an outer_function() that defines a global variable global_variable and a local variable local_variable. Inside the outer_function(), we have an inner_function() that modifies the local_variable using the nonlocal keyword, which allows access to the variable in the outer function's scope.

def outer_function(): global global_variable local_variable = "I'm a local variable" def inner_function(): nonlocal local_variable local_variable = "Modified local variable" print(local_variable) inner_function() print(local_variable) print(global_variable) global_variable = "I'm a global variable" outer_function() print(global_variable)
Global variables

In Python, global variables are not associated with a global object like in JavaScript. Instead, global variables in Python are accessible throughout the module or script in which they are defined. They can be accessed directly by their variable name.

In this code, global_variable is defined at the top level of the script, making it a global variable. Within the my_function(), the global variable global_variable can be accessed directly without any special syntax.

Constants

In Python, there is no specific keyword or syntax to declare constants like the const keyword in JavaScript. However, it is a common convention in Python to use uppercase variable names to indicate that a variable is intended to be treated as a constant. By convention, these constants are not meant to be changed once assigned.

PI = 3.14

In this case, PI is written in uppercase to indicate that it is a constant value representing the mathematical constant Pi. While Python allows you to modify the value of a variable, it is customary to treat variables written in uppercase as constants and avoid changing their values throughout the execution of the program.

Python does not enforce immutability or prevent reassignment of variables, including those declared in uppercase. However, it is generally considered a best practice and a convention within the Python community to treat variables declared in uppercase as constants and refrain from modifying their values.

It's worth noting that Python does not provide built-in mechanisms to enforce constant behavior, such as preventing reassignment or generating errors if a constant is modified. It is up to the programmer to follow the convention and treat variables declared as constants as read-only throughout the codebase.

Data types

Python also has several data types that you can use to work with different kinds of values. Here are the commonly used data types in Python:

  • Boolean: Represents the truth values True and False. It is used in logical operations and conditionals.
  • None: Similar to null in JavaScript, None represents the absence of a value. It is often used to indicate the absence of a return value or an uninitialized variable.
  • Numeric Types:
  • String: Represents a sequence of characters enclosed in single quotes (' ') or double quotes (" "). Strings are used to store and manipulate textual data.
  • List: Represents an ordered collection of items, enclosed in square brackets [ ]. Lists can contain elements of different types and can be modified (mutable).
  • Tuple: Similar to a list, a tuple represents an ordered collection of items, enclosed in parentheses ( ). However, tuples are immutable, meaning they cannot be modified after creation.
  • Set: Represents an unordered collection of unique elements. Sets do not allow duplicate values and provide methods for set operations like union, intersection, and difference.
  • Dictionary: Represents a collection of key-value pairs enclosed in curly braces { }. Each key-value pair maps a unique key to its associated value. Dictionaries are used for efficient lookup and retrieval of values based on keys.
if...else statement
In Python, the if statement is used to conditionally execute a block of code based on a logical condition. The syntax of the if statement in Python is as follows: if condition: statement_1 else: statement_2

The condition can be any expression that evaluates to either True or False. If the condition evaluates to True, statement_1 is executed. Otherwise, if the condition evaluates to False, statement_2 is executed.

x = 5 if x > 0: print("x is positive") else: print("x is non-positive") x = 10 if x > 0: print("x is positive") elif x == 0: print("x is zero") else: print("x is negative")
while statement
In Python, the while statement is used to repeatedly execute a block of code as long as a specified condition evaluates to true. The syntax of the while statement in Python is as follows: while condition: statement

The condition is an expression that is evaluated before each iteration of the loop. If the condition evaluates to True, the statement block is executed. After the execution of the statement block, the condition is evaluated again. If the condition is still True, the statement block is executed again. This process continues until the condition becomes False

Here's an example that demonstrates the usage of the while statement in Python:

n = 0 x = 0 while n < 3: n += 1 x += n print("Final values - n:", n, "x:", x)

In this example, the while loop iterates as long as the value of n is less than 3. Inside the loop, n is incremented by 1, and its value is added to x in each iteration. The loop continues until the value of n reaches 3. After completing the third iteration, the condition n < 3 evaluates to False, and the loop terminates.

Function declarations

In Python, a function definition begins with the def keyword, followed by the name of the function and a list of parameters enclosed in parentheses. The function block is indicated by a colon (:) and is comprised of Python statements indented below the function definition.

def square(number): return number * number

In this example, the function square is defined with a single parameter called number. The function body consists of a single statement that calculates and returns the square of the input number using the return keyword.

result = square(5) print(result) # Output: 25

In this code snippet, the square function is called with the argument 5. The returned result, which is the square of 5, is assigned to the variable result. Finally, the print statement displays the value of result, resulting in an output of 25.
